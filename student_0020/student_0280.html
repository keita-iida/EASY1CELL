To support this hands-on tutorial, I’ve prepared a small toolset that virtually generates minimal raw scRNA-seq data.

You’ll find it in the file MiniDrop.zip, which I’m pleased to offer as a gift for participants of this tutorial. It’s designed to help you learn by simulating the Drop-seq process from start to finish.

  • MiniDrop.zip

Required tools

To run MiniDrop, you’ll need a Linux or Linux-like environment with the following tools installed (unfortunately, MacOS didn't work in my pretest, which is underconstruction):

  • samtools
  • Picard
  • Dropseq Tools
  • STAR
We recommend using Conda to manage installation and dependencies easily.
